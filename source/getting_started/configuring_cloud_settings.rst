Configure the cloud settings 
============================

To configure your cloud settings and integrate Labiteers with Microsoft Azure:

1) Click your name in the bottom-left corner and select **Cloud Settings**.

2) Fill in the required fields and select **Save Settings**.

    .. image:: ./images/configure_cloud_settings.png
       :alt: Configure cloud settings
       :width: 60%

3) Verify your connection with Microsoft Azure by selecting **Test Connection**.

    .. image:: ./images/test_connection.png
       :alt: Test connection
       :width: 60%
