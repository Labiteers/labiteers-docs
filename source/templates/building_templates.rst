How to build a template 
=======================

1) To build a template, go to **List Resources**, select the virtual machine you want to use, and click **Build Template**.

    .. image:: ./images/build_template.png
       :alt: Build a template
       :width: 100%

2) Assign a meaningful name to your template and click **Build Template**. 

    .. image:: ./images/build_template2.png
       :alt: Name a template
       :width: 100%

3) The newly created template will then appear under **Templates**.

    .. image:: ./images/build_template3.png
       :alt: View a template
       :width: 100%
